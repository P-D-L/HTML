Frivillig fördjupning

Enklast är nog att du frågar om dessa uppgifter. Det är ganska simpelt men skulle kräva en hel del text och figurer för att förklara.

  1. Elektronen bestäms av fyra kvanttal, n, l, m och ms. (Detta innebär grovt sett att n är skalets nummer, l skalets form, m en undernivå i denna form och ms det s.k. spinnkvanttalet.) Man har funnit följande regler för dessa:
     
    n = 1, 2, 3, 4, ...
    l = 0, 1, 2, ..., n – 1
    m = –l, ..., –2, –1, 0, 1, 2, ... l
    ms     = –1/2, +1/2
     
    Enligt Pauliprincipen kan två elektroner aldrig ha alla fyra kvanttalen lika och därför fylls elektronerna på i atomen i en viss ordning. Det senare är "orsaken" till periodiska systemet. Av historiska skäl ger man också kortare namn åt olika elektronnivåer genom att ersätta l med bokstäver s, p, d, f, g (för 0, 1, 2, 3, 4).
    1. Fortsätt fyll i tabellen efter stigande kvanttal. (Eller kanske ännu hellre om du kan programmering; skriv ett litet program som genererar tabellen.)
      n l m ms Namn
      1 0 0 –1/2 1s
      1 0 0 +1/2 1s
      2 0 0 –1/2 2s
      2 0 0 +1/2 2s
      2 1 –1 –1/2 2p–1
      2 1 –1 +1/2 2p–1
      2 1 0 –1/2 2p0
      2 1 0 +1/2 2p0
      2 1 1 –1/2 2p1
      1 1 1 +1/2 2p1
      ... ... ... ... ...

    2. En enkel regel som först användes av Bohr säger att energin ökar efter summan n + l och efter ökande n. Använd denna regel för att försöka avgöra elektronkonfigurationen för några olika atomer i olika delar av det periodiska systemet.
