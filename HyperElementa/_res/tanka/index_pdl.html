Interaktiva meddelanden
Netscape Communicator
  • Webbläsare från Netscape Communications Corporation.
  • En fullständig installation av version 4.7 eller senare räcker troligen.
Internet Explorer
  • Webbläsare från Microsoft Corporation.
  • En fullständig installation av version 5.0 eller senare räcker troligen.
ShockWave Flash
  • Insticksmodeul ("plugin") från Macromedia för animeringar och interaktivitet.
Acrobat Reader
  • Insticksmodul eller fristående program från Adobe Coropration för att visa dokument i pdf.
  • Används bara i något enstaka fall i detta läromedel så du kan eventuellt klara dig utan så länge.
QuickTime
  • Insticksmodul eller fristående program från Apple Computer för att visa film.
